An HTTP message is the simply the formal conversation that happens between the web browser(the client) and the web server to ecxchange data.

Consists of two parts
    - Request(order request): asking the server for something(eg: "please show me the message")

    - Response(the waiter's delivery): server replying to the request (eg: "here is the webpage data")

Key components of the message:

    - start line : defins the action or results
    - headers : extra info about the request
    -empty line: divider that seperates the header from the body
    - body: the actual content being sent
